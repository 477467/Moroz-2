Основні команди Git

git init
Ініціалізація нового Git репозиторію
git status
Перевірка статусу файлів у репозиторії
git add .
Додавання всіх файлів до індексу
git add filename
Додавання конкретного файлу до індексу
git commit -m "message"
Створення коміту з повідомленням
git log
